# HNU SE Notes

这里是我在湖南师范大学软件工程专业的课程笔记整理仓库。  
从大一到大三，把每门课拆成独立文件夹，记录课堂内容、习题、复盘，以及一些个人理解。

这个仓库主要是给自己做长期的学习归档，也希望以后回头看时，还能记得当时的思考。

---

## 📂 目录结构

笔记按“学年 / 学期 / 课程”组织，例如：

docs/
├── y1-fall/ # 大一上（学期导航）
├── y1-spring/ # 大一下（学期导航）
├── courses/
│ ├── calculus-a-1/
│ │ ├── index.zh.md
│ │ ├── notes/
│ │ └── images/
│ ├── programming-basics/
│ └── ...
└── assets/

yaml
复制代码

课程都放在 `docs/courses/` 下，每门课一个独立文件夹，  
图片、PDF、Word 也归档到该课程文件夹内部，方便管理。
