# 日语备考（JLPT）

本人是国庆返校后才开始正式备考的……24年底开始上过一段时间线上课（n5没学完的程度），后来觉得讲个速度不适合自己就开始自学了，然后就是准备托福，暂停了学习。暑假接着学，但是当时忙着保研，也没学多少，暑假结束才把大家的日本语2学完，然后开始学新完全掌握。九月弃保后开始萎靡不振，但好歹把2给学完了，然后十月国庆后才开始正式学习新完全掌握3，匆匆忙忙吞完后，十月底开了真题卷━((*′д｀)爻(′д｀*))━!!!!

后期基本就是：  **背词 + 刷语法 + 做真题 + 反复复盘错误类型**  


---

## 📘 词汇（Vocabulary）

词汇是我个人最头疼的一环，主要以前完全没接触过，上半年也没好好背（哈哈就这样溺爱老己）只能靠反复出现来硬磨。

### 我的做法
- 不追求一次性记住
- 多轮刷，接受遗忘
- 重点放在 **“见到能反应出来”**，而不是默写完美

## 📗 语法（Grammar）

N2 语法的特点是：  
**单看不难，但混在一起就容易选错。**

### 我的做法
- 先系统过一遍语法点
- 再通过真题反复确认“它会怎么考”
- 特别容易混的语法单独摘出来反复看

## 🗣 敬语 / 即时应答

这一块是我觉得 **最需要单独拎出来练** 的部分，  
尤其是敬语，经常“看得懂但不会用”。

### 我的做法
- 单独集中练，不和其他语法混在一起
- 把常见句式当“模板”记
- 即时应答主要靠反应速度，多看多选
