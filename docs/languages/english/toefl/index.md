# TOEFL 托福备考

（DRN）波珠并不是语言天赋者，甚至可以说学的比较费力，以往做题都是靠应试技巧，所以备考时其实蛮痛苦的（呆瓜一个阿喂）。学托福也主要是为了以后能够出去看看这一需求所在，并不是对自己的实力有多么自信〒▽〒

这一页主要记录我对托福整体的理解，以及各个模块更适合怎样去练。

ps：听说托福又有变化了，此篇笔记仅总结2025/4前备考的一些心得。

---

## ⭐ 总体想法与备考定位

托福不一样的点首先是考试形式，不同于纸笔，机考无法做到写写画画，像我（多动症儿童）就很难集中精力，故多多限时在电脑上刷题是很好的！刷题网站我用的是**[学而思]([https://www.kmf.com/])**（打钱）），很好用！而且评论区也好有意思！

然后考前一周买了留星账号，刷网站上还没更新的真题。最后几天在**X大仙机经**（公众号）看了一下口语和写作的机经（老师推荐的）。

我一开始该怎么学，然后当时安排完时间线后，发现同期年底还要考日语/(ㄒoㄒ)/~~于是为了能尽可能快的考过就去机构上的，每天都有老师帮默单词+push作业并且跟进度，我觉得还是蛮不错的，就是还要同步上学有点赶(；′⌒`)。但我属于是考前一个月才开始高强度训练的（每天背默单词+练习），然后模考了几次，当然这期间还要同步上课+赶项目，有点太燃尽了，不建议大家也这么干(￣﹃￣)

如果已经具备一定英语基础，托福真正的难点感觉主要是 **在高信息密度下，无法稳定输出和判断重点**。所以多练，然后逐渐找到适合自己的训练方式和做题的手感还是很重要的。

---

## 📘 听力（Listening）

我一开始属于是词也听不懂的程度，于是就不停的去做精听精写，然后去看什么单词没听出来，但起色不太大（因为我也没有好好背单词。。。）后来发现听力部分的关键并不是听清每一个单词，而是跟上讲座或对话的**结构走向**。其中最重要的就是抓住关键的结构词，以及分清楚整个听力中的逻辑关系，所以后期精听的话，懂意思的单词和句子不用再反复誊抄，浪费时间，直接上框架和关键词就行。

ps：听力时的笔记不是越多越好，建议学一下口译中的符号表达[口译笔记法.pdf](https://huggingface.co/datasets/Deerainy/HNU-SE-materials/resolve/main/toefl/口译笔记法.pdf)

## 📗 阅读（Reading）

阅读如果逐字理解会非常吃力，更有效的方式是**先识别题型，再定位信息**。我以前从初中开始就喜欢通篇阅读后再做题，因为做题快热切难度也不是很大，所以ok，但是托福的阅读量大，时间有限，并且内容并不简单，故通篇阅读下来不仅耗时而且忘掉的概率很大，彼时再去定位的话很难顺利做完。

比较好的是，他的题目和所在段落大多都会标注出来，而且是顺序的，故顺着仔细边读边做就ok。然后就是要在平时训练注意力的集中，要一直盯着屏幕硬看那么久还是挺难的（于我而言）

平时训练的话，阅读复盘仔细一点比较好，可以先边抠生词，边翻译，然后再一句话去概括这一段的意义/意思/作用，然后再去刷一遍题，可以参考[阅读复盘案例-DRN.pdf](https://huggingface.co/datasets/Deerainy/HNU-SE-materials/resolve/main/toefl/阅读复盘案例-DRN.pdf)

## 🗣 口语（Speaking）

口语主要是在极短时间内完成一次结构完整的输出**，首先第一重要的是在听的过程中尽可能多的记录关键词，这和听力不太一样，然后就是不要害怕开口练习，波珠就经常骑车以及洗澡的时候自言自语（bushi

平时在背默单词的时候，要注意发音。ps：看美剧/英剧很有效果！我看了就有想开口表达的冲动！备考期间就这样看完了《生活大爆炸》全部。。。
