# 语言学习｜模块总览

> 本模块用于汇总与导航语言学习相关资料，涵盖不同考试方向与学习路径，供参考与交流使用。

---

## 🌐 模块说明

语言学习部分以**个人整理资料 + 后续共同补充**为主要形式构建。  
目前内容主要集中在英语考试方向，后续将根据实际情况逐步扩展。

需要说明的是：

- **托福（TOEFL）相关资料**主要由我本人（DRN）整理与上传  
- **雅思（IELTS）方向**后续可能会邀请或收录其他已经参加过考试的同学，补充他们的经验贴与复习思路  
- 不同资料来源、备考背景与目标分数可能存在差异，请结合自身情况选择参考
